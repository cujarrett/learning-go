# Concurrency

This module demonstrates core Go concurrency patterns through focused, standalone examples.

## Key Concepts

- **Goroutines and Channels**: Using goroutines for concurrent execution and channels for communication.
- **Worker Pools**: Limiting the number of concurrent workers to control resource usage.
- **Context Cancellation**: Using context to cancel operations and handle timeouts.
- **Buffered Channels**: Using buffered channels to reduce blocking in producer/consumer workflows.
